*상수
한번 설정해두면 그 프로그램이 종료 될 때까지 결코 변경될 수 없는 값
상수를 직접 나타내는 것보다 이름을 붙여 사용하는 것이 좋다.
상수이름은 대문자로 관례적으로 사용

#define 상수이름 상수값 - 매크로 상수로 정의(전처리, 컴파일되지 않음)

const 자료형 상수이름 = 상수값;

#define PI 3.141592

const double PI= 3.141592;

//PI = 3.14(변경할 수 없다.)


*연산자

항 : 연산에 사용되는 값

% : 앞 항에서 뒤 항을 나누어 나머지를 구한다.

관계연산자 : 연산 결과가 참이면 1, 거짓이면 0을 반환
== : 두개의 항 값이 같으면 참, 아니면 거짓을 반환합니다.
!= : 두개의 항 값이 다르면면 참, 아니면 거짓을 반환합니다.

논리연산자
&&(논리 곱) 두 항이 모두 참인 경우에만 결과 값이 참입니다.
||(논리 합) 두 항 중 하나의 항만 참이면 결과 값이 참입니다.
! 단항 연산자, 참인 경우는 거짓으로, 거짓인 경우는 참으로 바꿉니다.

비트연산자 -> 이진수

조건연산자 (조건식?결과1:결과2;) : 참이면 결과1, 거짓이면 결과2
